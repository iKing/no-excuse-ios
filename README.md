# NO EXCUSE

一个完全离线的 iPhone 原生执行强制器。任何时刻只允许一个行动存在，用 25 分钟让想法接触现实。

## 技术栈

- SwiftUI
- SwiftData
- iOS 17+
- 无服务器、无登录、无第三方依赖

## 运行

1. 使用 Xcode 打开 `NOEXCUSE.xcodeproj`。
2. 在项目的 Signing & Capabilities 中选择你的 Apple Developer Team。
3. 连接已启用开发者模式的 iPhone，选择设备后点击 Run。

Bundle Identifier 默认为 `com.iking.noexcuse`；如与你的签名账户冲突，请改为你拥有的唯一标识符。
